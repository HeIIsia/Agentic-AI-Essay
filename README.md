# Agentic-AI-Essay
A simple agentic workflow designed to simulate reflective thinking in a writing task.

### Step 1 – Drafting: 
Call the LLM to generate an initial draft of an essay based on a simple prompt.

### Step 2 – Reflection: 
Reflect on the draft using a reasoning step.

### Step 3 – Revision: 
Apply the feedback from the reflection to generate a revised version of the essay.
